1.1. Example 1: Grab metadata from a database¶

In this example, we will learn how to get metadata of each file in a database. The metadata contains objective information (e.g. recording date, duration, etc.) as well as subjective information such as tags.

1.1.1. List database IDs¶

Firstly, let’s import V3DBHandler from the toolkit to list the ID of databases in example.db.

[1]:

from pydtk.db import V3DBHandler as DBHandler

db_id_handler = DBHandler(db_class='database_id', db_host='./example.db')
db_id_handler.df

[1]:
database_id df_name creation_time_in_df
uuid_in_df
c21f969b5f03d33d43e04f8f136e7682 default db_0ffc6dbe_meta 1.603886e+09

You can see that example.db contains a database with ID default.

1.1.2. Grab a DataFrame of metadata from a database¶

Now, let’s initialize another V3DBHandler to retrieve the metadata of the contents in default database.

[2]:

from pydtk.db import V3DBHandler as DBHandler

db_handler = DBHandler(
    db_class='meta',
    db_host='./example.db',
    database_id='default',
    base_dir_path='../test',
    read_on_init=True
)
If you set read_on_init to True, the entire contents in the database will be loaded and stored into the local memory as a Pandas DataFrame.

When you want to handle a very large dataset, the metadata contains huge amount of information and as a result, it takes a long time to load all of it.
However, if you want to grab only a limited scope (e.g. metadata of files tagged ‘camera’ and ‘front’), it is costful to load all the dataset and search items on the loaded dataframe.
Therefore, the toolkit provides a method to execute a sql query before loading the database and limit the items to load.

To execute a sql query before loading metadata, you should set read_on_init option to False as follows.

[4]:

db_handler = DBHandler(
    db_class='meta',
    db_host='./example.db',
    database_id='default',
    base_dir_path='../test',
    read_on_init=False
)
db_handler.read(where='start_timestamp > 1500000000')
print('# of metadata: {}'.format(len(db_handler.df)))
db_handler.read(where='tags like "%camera%" and tags like "%front%"')
print('# of metadata: {}'.format(len(db_handler.df)))
db_handler.read(where='tags like "%can%" and tags like "%steering%"')
print('# of metadata: {}'.format(len(db_handler.df)))
db_handler.read(where='tags like "%can%" or tags like "%camera%"')
print('# of metadata: {}'.format(len(db_handler.df)))

# of metadata: 8
# of metadata: 1
# of metadata: 0
# of metadata: 2
